# 组价挂载（引用）

## 一、直接使用

```js
<template>
  <div>
    <h1>主组件</h1>
    <FirstComp />
  </div>
</template>

<script>
import FirstComp from "./FirstComp";
export default {
  components: {
    FirstComp,
  },
};
</script>

<style></style>

```

- 引用的时候，FirstComp可以写成`first-comp`，具体看场景（风格指南）。
