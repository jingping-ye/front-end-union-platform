# 监听器

## 一、特点

一个值的变化会引起其他值的变化

## 二、声明方式

1. 简单声明方式

```js
<template>
  <div>
    <div>{{ str1 }}</div>
    <div><button @click="clickMe">点击我</button></div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      str1: "原来的文本",
    };
  },
  watch: {
    str1(newV, oldV) {
      console.log("===newV===", newV);
      console.log("===oldV===", oldV);
    },
  },
  methods: {
    clickMe() {
      this.str1 = "改变的文本";
    },
  },
};
</script>

<style></style>


===== 控制台输出 =====
    
===newV=== 改变的文本
===oldV=== 原来的文本

===== 控制台输出 =====
```

- 默认两个参数，一个是新值一个是旧值。

2. 完全声明方式

```js
<template>
  <div>
    {{ person.name }}
    <button @click="clickMe">点击我</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      person: { name: "joy" },
    };
  },
  watch: {
    person: {
      handler(newV, oldV) {
        console.log("newV", newV);
        console.log("oldV", oldV);
      },
      deep: true,
      immediate: true,
    },
  },
  methods: {
    clickMe() {
      this.person.name = "mark";
    },
  },
};
</script>

<style></style>

```

- `deep`：监听对象中所有属性的变动，及时对象的嵌套层次很深。
- `immediate`：立即监听。监听器默认在初始化时不监听，只在初始化后值改变后才会监听。

## 三、其他

### 1. 监听对象的某一项

> - 这里仍然是仿照上面的监听人名功能
> - 使用点访问符指定对象的属性，那么即可监听某一项。

```js
<template>
  <div>
    {{ person.name }}
    <button @click="clickMe">点击我</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      person: { name: "joy" },
    };
  },
  watch: {
    "person.name": {
      handler(newV, oldV) {
        console.log("newV", newV);
        console.log("oldV", oldV);
      },
    },
  },
  methods: {
    clickMe() {
      this.person.name = "mark";
    },
  },
};
</script>

<style></style>
```

### 2. 监听实例中某一项

> 实例中的属性皆可以进行监听，要了解实例中有什么属性，可以使用`console.log(this)`的方式输出查看。

1. 监听实例中路由变动

   ````js
   <template>
     <div>
       <button @click="clickMe">点击我</button>
     </div>
   </template>
   
   <script>
   export default {
     watch: {
       $route: {
         handler(newV, oldV) {
           console.log("newV", newV);
           console.log("oldV", oldV);
         },
         immediate: true,
       },
     },
     methods: {
       clickMe() {
         console.log("this", this);
       },
     },
   };
   </script>
   
   <style></style>
   
   ````

2. 监听实例中的数据变动

   ```js
   <template>
     <div>
       <button @click="clickMe">点击我</button>
     </div>
   </template>
   
   <script>
   export default {
     data() {
       return {
         str: 1,
       };
     },
     watch: {
       $data: {
         handler(newV, oldV) {
           console.log("newV", newV);
           console.log("oldV", oldV);
         },
         deep: true,
       },
     },
     methods: {
       clickMe() {
         this.str = 2;
         console.log("this", this);
       },
     },
   };
   </script>
   
   <style></style>
   ```

### 3. 防抖与节流

- 防抖(debounce)：短时间内事件如果被重复触发，则不执行，一直到事件不再在短时间内被触发。
- 节流（throttle）：一定时间内执行一次。

未做节流与防抖前：

```js
<template>
  <input v-model="str" />
</template>

<script>
export default {
  data() {
    return {
      str: "",
    };
  },
  watch: {
    str(newV, oldV) {
      console.log("newV", newV);
      console.log("oldV", oldV);
    },
  },
};
</script>

<style></style>


//	输入12345进行测试
==== 控制台输出 =====
newV 1
oldV 
newV 12
oldV 1
newV 123
oldV 12
newV 1234
oldV 123
newV 12345
oldV 1234
==== 控制台输出 =====
```

做防抖处理：

```js
<template>
  <input v-model="str" />
</template>

<script>
let timer = null;
export default {
  data() {
    return {
      str: "",
    };
  },
  watch: {
    str(newV, oldV) {
      if (timer) {
        clearTimeout(timer);
        timer = null;
      } else {
        timer = setTimeout(() => {
          console.log("newV", newV);
          console.log("oldV", oldV);
        }, 75);
      }
    },
  },
};
</script>

<style></style>


//	输入12345进行测试
==== 控制台输出 =====
newV 1
oldV 
newV 123
oldV 12
newV 12345
oldV 1234
==== 控制台输出 =====

```

做节流处理：

```js
<template>
  <input v-model="str" />
</template>

<script>
let delay = delay || 200; // 设置200ms执行一次
let lastTime = null;
export default {
  data() {
    return {
      str: "",
    };
  },
  watch: {
    str(newV, oldV) {
      let nowTime = Date.now();
      if (lastTime && nowTime - lastTime > delay) {
        console.log("newV", newV);
        console.log("oldV", oldV);
      } else {
        lastTime = nowTime;
      }
    },
  },
};
</script>

<style></style>

==== 控制台输出 =====
newV 1234
oldV 123
newV 12345
oldV 1234
==== 控制台输出 =====
```
