# Vue CLI初始项目结构分析

```text
|-- node_modules // npm安装的项目包
|-- public // 项目打包的一个配置目录
	|-- favicon.ico // 网站的图标
	|-- index.html  // 网站的入口页面
|-- src // 开发源文件 --> 你要找的开发文件都在这里
	|-- assets // 图片资源目录
	|-- components // 公共组件
	|-- App.vue vue文件的入口文件
	|-- main.js // js的入口文件
|-- .gitignore // git忽略文件
|-- babel.config.js // babel配置文件
|-- package-lock.json // 包项目详细信息文件，包括主包和子包等信息。
|-- package.json // 包管理文件
|-- README.md 项目描述文件
```
