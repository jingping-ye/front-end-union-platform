# 四、脚手架

Vue-CLI

  1. 安装
npm install @vue/cli -g
  1. 快速原型开发
# 安装
npm install -g @vue/cli-service-global

# 监听 Hello.vue文件
vue serve ./Hello.vue
  1. 正式项目
# 自动创建一个project-name文件夹,里面是项目文件
vue create project-name

# 直接在当前文件夹下生成项目文件
vue create .
  1. 安装插件
# 安装vue-router
vue add router
