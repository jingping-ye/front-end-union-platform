# 分包

小程序大于2M时，需要将小程序进行分包

# 简单的分包

分包后，小程序分为普通包和子包。

注意：

  1. 子包可以调用普通包中的东西，普通包不可以调用子包的东西
  2. 路径：由于进行了分包，如果是先开发后分包的话，要注意路径。从普通包到子包要加上路径/pages/subpack/pages/index/index,子包使用普通包的东西也要修改路径。
  3. 子包里面的东西刚进来小程序时并不会加载，只有触及了才会加载。
